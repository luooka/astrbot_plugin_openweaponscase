# astrbot_plugin_openweaponscase

cs武器箱开箱模拟，可以展示开启物品的图案和对应的磨损  
## 指令

`开箱菜单`: 介绍菜单  
`开箱`: 开箱[武器箱名称] 批量开箱：开箱[次数][空格][武器箱名称] 或 开箱[武器箱名称][次数]  
`库存`: 开启的历史库存  
`清除库存`: 清除历史库存信息
